多端小程序开发，用 taro 还是 uniapp？

发表于2021-06-08|更新于2021-06-08|Vueuniapp
|阅读量:

taro vs uniapp

uniapp taro
跨端平台 微信/支付宝/百度/头条/QQ/钉钉/淘宝/H5 微信/京东/百度/支付宝/字节跳动/QQ 小程序/H5

总结：uview-ui 组件最全，但是安卓手机上问题不少。

总结

用 taro 还是 uniapp 这个问题，不能只看技术栈，还要结合公司自身的优势，如果公司没有会使用 react 的开发人员，那就建议选 uniapp，比如我所在的公司。

至于如何选 ui 库，我建议 uniapp 用 uni-ui，taro 用 taro-ui。
