五种方式优缺点总结

方式 优点 缺点
使用 fetch 无论什么文件都是执行下载 fetch 本身的兼容性问题
基于 axios 封装的 http 请求 无论什么文件都是执行下载 不适合大文件下载
window.open 简单快速 需要同域、可能会被拦截、浏览器支持预览的文件不会直接下载
form 表单提交 兼容性好 浏览器支持预览的文件不会直接下载
a 标签触发 兼容性好 可能会被拦截、浏览器支持预览的文件不会直接下载
