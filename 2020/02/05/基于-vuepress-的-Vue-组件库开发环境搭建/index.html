基于 vuepress 的 Vue 组件库开发环境搭建

发表于2020-02-05|更新于2021-04-13|Vue
|阅读量:

vuepress 是什么？引用官方的一句话：Vue 驱动的静态网站生成器。适合用来做文档类型的网站搭建，基于它我搭建了一个 Vue 组件库的开发环境。

特性

vuepress 自身提供的能力是强大的，这也意味着有比较多的 API 和各种繁琐配置项，如果你正在找 Vue 组件库的脚手架，可以作为一个参考，本脚手架具有以下几个特点：

1、开箱即用，不用过多的关注配置，专注于自身的组件开发

2、 文档格式参照 vuepress 官网，不是很好看，也不算太丑。。。

3、组件使用 @vue/cli 一键打包，方便快捷

效果如下：

完整效果查看，请移步这里，或者通过全局安装 bs-cli，然后执行 bs init，选择「组件库(Vue)」这个选项创建模板来查看。

使用

新增组件只需要三步：

1、在 src 里面新增组件

2、在 docs/.vuepress/components/demo 新增使用姿势

3、在 docs/components 里面新增使用文档

至于细节，vuepress 官方文档描述得非常详细，这里就不废话了，传送门。
